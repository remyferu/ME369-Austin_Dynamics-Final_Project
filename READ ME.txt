Install ROS and turtlebot3 sim
Download the workspace from github
Open terminal
run commands:

$ cd ~/ME369P/Project
$ source devel/setup.bash
$ export TURTLEBOT3_MODEL=waffle_pi
$ roslaunch turtlebot3_gazebo turtlebot3_final_project.launch
gazebo will open

In a new terminal, run command:

$ cd ~/ME369P/Project/src/final_project
$ chmod +x model_mover.py
$ cd ~/ME369p/Project
$ source devel/setup.bash
$ rosrun final_project model_mover.py
